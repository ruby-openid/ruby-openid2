Ruby OpenID2 Library Installation

Install as a gem

ruby-openid2 is distributed on RubyGems.
Install it:

gem install ruby-openid2

Testing the Installation

Make sure everything installed ok:

$> irb
irb$> require "ruby-openid2"
=> true
